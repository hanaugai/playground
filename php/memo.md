# PHP

## Overview

プログラミング言語としての特徴
- HTML埋め込み型の構文（Hypertext Preprocessorたる所以）
- 弱い動的型付け
- クラスベースオブジェクト指向のサポート
- 例外処理 (try, catch, throw) のサポート

処理系としての特徴
- サーバーサイドWebアプリケーション構築のための豊富な組み込み関数
- データベースへの容易なアクセス（ベンダーごとの組み込み関数、PDO）
- PECLによる言語機能の拡張
- 多くのWebサーバへの組み込みの標準サポート
